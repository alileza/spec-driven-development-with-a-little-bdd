# Spec-Driven Development with a Little Bit of BDD

A presentation exploring spec-driven development and behavior-driven development approaches for working with AI coding agents.

## Links

- **Live Presentation**: https://alileza.github.io/spec-driven-development-with-a-little-bdd/
- **GitHub Repository**: https://github.com/alileza/spec-driven-development-with-a-little-bdd

## About

This presentation covers:
- The evolution from AI as a "PoC tool" to a practical development assistant
- How spec-driven development helps AI produce consistent, quality code
- The challenges of maintaining specs and how BDD provides executable specifications
- Practical workflows using tools like GitHub's Spec-Kit and Tomato BDD framework

**Speaker**: Ali Reza Yahya (@alileza)
**Duration**: ~15-20 minutes
